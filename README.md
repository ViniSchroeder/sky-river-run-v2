# Sky River Run — Versão 1.2

Jogo arcade de avião feito com HTML5 Canvas e JavaScript puro.

## Recursos atuais

- Rio sinuoso e estreitamentos.
- Limite físico nas margens.
- Combustível e galões de reposição.
- Bombas para destruir pontes.
- Pássaros, asas-deltas, helicópteros e discos voadores.
- Pontuação diferente por inimigo.
- Tiro simples acionado por comando.
- Helicóptero libera tiro duplo.
- Disco voador libera tiro triplo.
- Sistema de três vidas.
- Nuvens de baixa visibilidade depois de 3.000 pontos.
- Controles para desktop e celular.
- PWA básica e cache offline.

## Controles

### Desktop

- WASD ou setas: pilotar
- Espaço: atirar
- B: usar bomba

### Celular

- Arraste sobre a tela: pilotar
- Botão de fogo: atirar
- Botão de bomba: usar bomba

## Publicação na Vercel

1. Extraia este ZIP.
2. No GitHub, clique em **uploading an existing file**.
3. Arraste todos os arquivos e pastas desta pasta para o repositório.
4. Confirme em **Commit changes**.
5. Na Vercel, importe o repositório.
6. Framework Preset: **Other**.
7. Não preencha comandos de build.
8. Clique em **Deploy**.

O arquivo `index.html` deve aparecer na raiz do repositório.


## Novidades da versão 1.2

- Tiro contínuo enquanto a barra de espaço ou botão de fogo estiver pressionado.
- Aviões, helicópteros, aves e discos voadores atravessando lateralmente.
- Barcos navegando verticalmente.
- Submarinos que emergem aleatoriamente.
- Caminhões atravessando pontes.
- Caminhões valem 160 pontos e melhoram o nível de tiro.
- Novo avião amarelo em estilo monomotor de desenho animado.
- Pontuação própria por tipo de inimigo.
