# Sky River Run 3.0 — Pixel Arcade

Reformulação visual completa em pixel art, mantendo a jogabilidade construída nas versões anteriores.

## Destaques
- Avião pixel art com sprites de curva.
- Água e margens em tiles pixelados.
- Inimigos, barcos, submarinos, caminhões e UFOs redesenhados.
- Tiros por nível com sprites próprios.
- Itens de combustível, escudo, reparo, bomba, moeda e vida.
- Explosão animada em 8 quadros.
- HUD arcade.
- Joystick translúcido no celular.
- Cache totalmente renovado.
